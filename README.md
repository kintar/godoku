## Godoku - A TUI Sudoku Game in Go

Godoku is a terminal-based Sudoku game written in Go. It provides an interactive and user-friendly interface for solving
Sudoku puzzles directly from the command line. Eventually, the game will support puzzle generation and multiple levels of
difficulty, along with hints and tutorials for solving Sudoku.

## Status

Very much work in progress. I'm in the process of proving out my TUI concepts, as well as setting up the GitHub
project to track milestones.
